HTML, or HyperText Markup Language, is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

The first version of HTML was written by Tim Berners-Lee in 1993. It has evolved significantly since then, with HTML5 being the latest major version.

CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, or in other media.

CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.
